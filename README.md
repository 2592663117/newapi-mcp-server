# NewAPI MCP Server

MCP Server for NewAPI image generation (gpt-image-2 model).

## Environment Variables

- `NEWAPI_BASE_URL`: Your NewAPI base URL (default: https://jiuuij.de5.net)
- `NEWAPI_API_KEY`: Your NewAPI API key

## Usage

This MCP server provides image generation tool via NewAPI.

## Config for ModelScope MCP Plaza

- Mode: Streamable HTTP
- Port: 8000
